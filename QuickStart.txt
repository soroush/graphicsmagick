ImageMagick now depends on an external file, delegates.mgk.  If certain
ImageMagick commands fails, check this file and verify it is valid for
your environment.  For details beyond this document, see README.txt
and ImageMagick.html.

Unix:

  Download

    ftp://www.wizards.dupont.com/pub/ImageMagick/ImageMagick-4.1.2.tar.gz

  and type

    gunzip -c ImageMagick-4.1.2.tar.gz | tar xvf -
    cd ImageMagick-4.1.2
    ./configure -enable-shared -enable-lzw -with-perl
    make
    make install
    convert -verbose logo: image.gif
    display image.gif

  Alternatively, download the appropriate binary from
  ftp://www.wizards.dupont.com/pub/ImageMagick/binaries.


NT:

  Download

    ftp://www.wizards.dupont.com/pub/ImageMagick/nt/ImageMagick-nt.zip

  and double click on it to extract the distribution.  Next, copy

    ImageMagick-4.1.2/ImageMagick-4.1.2/delegates/nt.mgk

  to

    c:\ImageMagick\delegates.mgk

  Select File->New Folder to create c:\ImageMagick.  Use Copy/Paste to
  copy delegates.mgk.

  If you have an X server, change to ImageMagick-4.1.2/ImageMagick-4.1.2/bin
  and double-click on display.exe.  Otherwise, select Start->Command Prompt.
  In the Command Prompt window type

    cd ImageMagick-4.1.2
    cd ImageMagick-4.1.2
    cd bin
    convert -verbose logo: image.gif
    convert image.gif win:

  Alternatively, download the ImageMagick NT source

    ftp://www.wizards.dupont.com/pub/ImageMagick/nt/ImageMagick-4.1.2.zip

  and double click on the ImageMagick Visual C++ project to build from
  the source.
