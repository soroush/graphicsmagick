ImageMagick ImageAttribute Structure

The ImageAttribute structure is used to add arbitary textual attributes to an image. Each attribute has an associated key and value. Add new attributes, or update an existing attribute, via SetImageAttribute() and obtain the value of an existing attribute via GetImageAttribute().  Key-strings used by ImageMagick include "Comment" (image comment), "Label" (image label), and "Signature" (image signature).

The members of the ImageAttribute structure are shown in the following table:
 

ImageAttribute Structure Members
Member
Type
Description
key char * Attribute key
value char * Attribute value
