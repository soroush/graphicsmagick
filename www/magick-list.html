The ImageMagick mailing lists are a low noise and subject oriented. The subject is the discussion of ImageMagick software and its use. Although the lists are unmoderated, do not post off-topic or test messages to the lists. Off-topic messages will result in the offender being silently removed from the lists and prevented from rejoining.

 
Subscribe to a list
E-mail the following text
    subscribe magick-list
in the body of a message to majordomo@imagemagick.org (don't place it in the Subject: field!). Where list is one of announce, developer, or user whichever is appropriate for you .  The purpose of the lists is as follows:
announce Announcements pertaining to ImageMagick, or ImageMagick related software. This list should not be used for any discussion.
user Questions and postings pertaining to the usage of ImageMagick regardless of the interface. This includes the command-line utilities, as well as the Perl, C++, and C APIs. Usage questions are like "How do I use ImageMagick to create drop shadows?".
developer Questions and postings pertaining to the development of ImageMagick, feature enhancements, and ImageMagick internals. ImageMagick source code and algorithms are discussed here. Usage questions which are too arcane for the normal user list should also be posted here.
(e.g. subscribe to the magick-developer list if you intend to participate in ImageMagick development discussions).
The mailing lists feed into each other. Users subscribed to the user list also receive postings to announce, while users subscribed to the developer list also receive postings to the announce and user lists.
 
Unsubscribe from a list
E-mail the following text
    unsubscribe magick-list
in the body of a message to majordomo@imagemagick.org (don't place it in the Subject: field!). Where list is one of announce, developer, or user
